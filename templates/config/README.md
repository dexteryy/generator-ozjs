## Getting started

### Require

1. node, npm
2. grunt v0.4 - `npm install grunt-cli -g`

4. ruby, gem, bundle

### Install dependencies

1. `npm install`
2. `bundle install`

### Build and test

1. `grunt`
2. `grunt connect`
3. see [http://localhost:9100](http://localhost:9001)
